# 触控方法命令文档

## 输入法相关

### `imeLib.deleteChar()`
- **功能**: 输入法删除一个字符
- **返回值**: 布尔值(true成功/false失败)
- **示例**:
```lua
imeLib.setText("hello nice!!!")
imeLib.deleteChar()
```

### `imeLib.finishInput()`
- **功能**: 输入法模拟完成输入
- **返回值**: 布尔值(true成功/false失败)
- **示例**:
```lua
imeLib.setText("hello nice!!!")
imeLib.finishInput()
```

### `imeLib.keyEvent(action, keycode)`
- **功能**: 输入法模拟按键事件
- **参数**:
  - `action`: 0(按下)/1(弹起)
  - `keycode`: Android KeyEvent键码
- **返回值**: 布尔值(true成功/false失败)
- **示例**:
```lua
imeLib.keyEvent(0, 7) -- 模拟按下字符"0"
imeLib.keyEvent(1, 7) -- 模拟弹起字符"0"
```

### `imeLib.setText(text)`
- **功能**: 输入法模拟输入文字
- **参数**:
  - `text`: 文本类型
- **返回值**: 布尔值(true成功/false失败)
- **示例**:
```lua
imeLib.setText("hello nice!!!")
```

## 输入法锁定

### `imeLib.lock()`
- **功能**: 锁定使用懒人输入法
- **返回值**: 布尔值(true成功/false失败)
- **示例**:
```lua
setStopCallBack(function(error)
	imeLib.unlock()
end)
imeLib.lock()
imeLib.setText("hello")
```

### `imeLib.unlock()`
- **功能**: 解锁懒人输入法
- **返回值**: 布尔值(true成功/false失败)
- **示例**:
```lua
setStopCallBack(function(error)
	imeLib.unlock()
end)
```

## 基础触控

### `inputText(text)`
- **功能**: 模拟输入文字
- **参数**:
  - `text`: 要输入的文本
- **返回值**: 布尔值(true成功/false失败)
- **示例**:
```lua
inputText("hello nice!!!")
```

### `tap(x, y)`
- **功能**: 点击坐标为x,y的点
- **参数**:
  - `x`: 横坐标
  - `y`: 纵坐标
- **示例**:
```lua
tap(100, 100)
```

### `longTap(x, y)`
- **功能**: 长点击坐标为x,y的点
- **参数**:
  - `x`: 横坐标
  - `y`: 纵坐标
- **示例**:
```lua
longTap(100, 100)
```

### `swipe(x1, y1, x2, y2, time)`
- **功能**: 模拟从一点滑动到另外一点
- **兼容性**: 所有模式可用
- **参数**:
  - `x1,y1`: 起点坐标
  - `x2,y2`: 终点坐标
  - `time`: 滑动时间(毫秒)
- **返回值**: 布尔值(true成功/false失败)
- **示例**:
```lua
swipe(200, 200, 800, 200, 500)
```

### `touchDown(id, x, y)`
- **功能**: 模拟手指按下
- **限制**: 不支持无障碍模式(仅激活/root模式)
- **参数**:
  - `id`: 手指索引(0-4)
  - `x,y`: 坐标
- **示例**:
```lua
touchDown(2, 100, 100)
touchDown(1, 100, 100)
sleep(300)
touchUp(1)
sleep(100)
touchUp(2)
```

### `touchMove(id, x, y)`
- **功能**: 模拟手指移动
- **限制**: 不支持无障碍模式(仅激活/root模式)
- **参数**:
  - `id`: 手指索引(0-4)
  - `x,y`: 坐标
- **返回值**: 布尔值(true成功/false失败)
- **示例**:
```lua
touchDown(1, 297, 327)
for i = 1, 300 do
    touchMove(1, 297, 327 - i)
end
touchUp(1)
```

### `touchMoveEx(id, x, y, time)`
- **功能**: 模拟滑动增强版
- **参数**:
  - `id`: 手指索引(0-4)
  - `x,y`: 坐标
  - `time`: 滑动时间(毫秒)
- **示例**:
```lua
touchDown(1, 100, 100)
sleep(50)
touchMoveEx(1, 300, 300, 800)
touchUp(1)
```

### `touchUp(id)`
- **功能**: 模拟手指抬起
- **限制**: 不支持无障碍模式(仅激活/root模式)
- **参数**:
  - `id`: 手指索引(0-4)
- **返回值**: 布尔值(true成功/false失败)
- **示例**:
```lua
touchDown(2, 100, 100)
touchDown(1, 100, 100)
sleep(300)
touchUp(1)
sleep(100)
touchUp(2)
```

## 按键相关

### `keyDown(keycode)`
- **功能**: 按下一个按键不弹起
- **限制**: 不支持无障碍模式(仅激活/root模式)
- **参数**:
  - `keycode`: 按键标识符或数字按键码
- **返回值**: 布尔值(true成功/false失败)
- **示例**:
```lua
keyDown("home")
sleep(100)
keyUp("home")
```

### `keyPress(keycode)`
- **功能**: 按一下按键并弹起
- **限制**: 无障碍模式仅支持home/back/recent/power
- **参数**:
  - `keycode`: 按键标识符或数字按键码
- **返回值**: 布尔值(true成功/false失败)
- **示例**:
```lua
keyPress("home")
```

### `keyUp(keycode)`
- **功能**: 弹起一个按键
- **限制**: 不支持无障碍模式(仅激活/root模式)
- **参数**:
  - `keycode`: 按键标识符或数字按键码
- **返回值**: 布尔值(true成功/false失败)
- **示例**:
```lua
keyDown("home")
sleep(100)
keyUp("home")
```

## 高级触控

### `setOnTouchListener(onTouchEvent)`
- **功能**: 监听用户触摸事件
- **限制**: 不支持无障碍模式(仅激活/root模式)
- **参数**:
  - `onTouchEvent`: 回调函数(x,y参数)
- **返回值**: 布尔值(true成功/false失败)
- **示例**:
```lua
function onTouchEvent(x, y)
    print(x, y)
end
setOnTouchListener(onTouchEvent)
sleep(1000000)
```
