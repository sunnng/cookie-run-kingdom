# 控制台方法文档

## 基本控制

### `console.show([isShow])`
- **功能**: 显示/隐藏控制台窗口
- **参数**:
  - `isShow`: true显示(默认), false隐藏
- **示例**:
```lua
console.show()  -- 显示控制台
console.show(false)  -- 创建但不显示
```

### `console.dismiss()`
- **功能**: 关闭控制台窗口
- **返回值**: 布尔值(成功/失败)

## 窗口配置

### `console.setPos(x,y,[w],[h])`
- **功能**: 设置控制台位置和大小
- **参数**:
  - `x,y`: 窗口坐标
  - `w,h`: 可选宽高
- **示例**:
```lua
console.setPos(100,100,800,600)  -- 设置位置和大小
```

### `console.setTitle(title)`
- **功能**: 设置控制台标题
- **参数**: `title` 标题文本

## 日志输出

### `console.println(level,log)`
- **功能**: 分级打印日志
- **参数**:
  - `level`: 日志级别(1-6对应VERBOSE-ASSERT)
  - `log`: 日志内容
- **示例**:
```lua
console.println(3, "重要信息")  -- INFO级别
console.println(5, "错误发生")  -- ERROR级别
```

### `console.clearLog()`
- **功能**: 清除控制台所有日志

## 高级控制

### `console.lockConsole()`/`unlockConsole()`
- **功能**: 锁定/解锁控制台窗口(防止误操作)
- **使用模式**:
```lua
console.lockConsole()
-- 执行关键操作
console.unlockConsole()
```

### `console.showTitle([isShow])`
- **功能**: 显示/隐藏标题栏
- **参数**: `isShow` true显示(默认), false隐藏
