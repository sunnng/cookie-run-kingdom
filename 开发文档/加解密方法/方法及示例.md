# 加解密方法文档

## AES加密

### `cryptLib.aes_crypt(data, key, operation, mode, [iv], [padding])`
- **功能**: AES对称加密/解密
- **参数**:
  - `data`: 原始数据(string)
  - `key`: 密钥(16/24/32字节对应AES-128/192/256)
  - `operation`: "encrypt"或"decrypt"
  - `mode`: 加密模式("ecb","cbc","cfb","ofb","ctr")
  - `iv`: 初始化向量(16字节，ecb模式不需要)
  - `padding`: PKCS#7填充(默认true)
- **返回值**: 加密/解密结果(string)
- **示例**:
```lua
-- AES-CBC加密
local encrypted = cryptLib.aes_crypt("data", "16bytekey12345678", "encrypt", "cbc", "16byteiv12345678")

-- AES-ECB解密
local decrypted = cryptLib.aes_crypt(encrypted, "16bytekey12345678", "decrypt", "ecb")
```

### `cryptLib.aes_ivgen()`
- **功能**: 生成随机初始化向量
- **返回值**: 16字节随机IV(string)

### `cryptLib.aes_keygen(key_length)`
- **功能**: 生成随机AES密钥
- **参数**: `key_length`: 密钥长度(16/24/32)

## RSA加密

### `cryptLib.rsa_encrypt(data, key, is_public_key)`
- **功能**: RSA加密/签名
- **参数**:
  - `data`: 原始数据(string)
  - `key`: PEM格式密钥
  - `is_public_key`: true使用公钥
- **注意**: 数据长度不能超过密钥长度-11字节

### `cryptLib.rsa_decrypt(data, key, is_public_key)`
- **功能**: RSA解密/验证签名
- **参数**:
  - `data`: 加密数据(string)
  - `key`: PEM格式密钥
  - `is_public_key`: true使用公钥
- **返回值**: 解密结果(string)

### `cryptLib.rsa_generate_key([key_bits])`
- **功能**: 生成RSA密钥对
- **参数**: `key_bits`: 密钥位数(默认2048)
- **返回值**: 公钥,私钥(PEM格式)
