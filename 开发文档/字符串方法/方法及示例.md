# 字符串方法文档

## 基础操作

### `string.byte(s)`
- **功能**: 获取字符ASCII码
- **示例**:
```lua
print(string.byte("A"))  -- 输出: 65
```

### `string.char(...)`
- **功能**: ASCII码转字符
- **示例**:
```lua
print(string.char(65,66,67))  -- 输出: ABC
```

## 字符串转换

### `string.lower(s)`/`string.upper(s)`
- **功能**: 大小写转换
- **示例**:
```lua
print(string.lower("ABC"))  -- 输出: abc
print(string.upper("abc"))  -- 输出: ABC
```

### `string.reverse(s)`
- **功能**: 字符串反转
- **示例**:
```lua
print(string.reverse("abc"))  -- 输出: cba
```

## 字符串查找

### `string.find(s, pattern)`
- **功能**: 查找子串
- **示例**:
```lua
local start, end = string.find("Hello Lua", "Lua")
print(start, end)  -- 输出: 7 9
```

### `splitStr(src, sep)`
- **功能**: 分割字符串
- **示例**:
```lua
local parts = splitStr("a,b,c", ",")
for i,v in ipairs(parts) do print(i,v) end
```

## UTF8处理

### `utf8.length(s)`
- **功能**: UTF8字符串长度
- **示例**:
```lua
print(utf8.length("中文"))  -- 输出: 2
```

### `utf8.mid(s, start, len)`
- **功能**: 截取UTF8子串
- **示例**:
```lua
print(utf8.mid("中国人民", 2, 2))  -- 输出: 人民
```

## 字符串格式化

### `string.format(...)`
- **功能**: 字符串格式化
- **示例**:
```lua
print(string.format("value is %d", 666))  -- 输出: value is 666
```

## 字符串长度

### `string.len(s)`
- **功能**: 获取字符串长度
- **示例**:
```lua
print(string.len("abcdefg"))  -- 输出: 7
```

## 字符串复制

### `string.rep(s, n)`
- **功能**: 复制字符串
- **示例**:
```lua
print(string.rep("abc", 2))  -- 输出: abcabc
```

## UTF8字符串操作

### `utf8.inStr(s, pattern)`
- **功能**: 查找UTF8子串
- **示例**:
```lua
local pos = utf8.inStr("中国人民万岁", "P")
print(pos)  -- 输出: 4
```

### `utf8.inStrRev(s, pattern)`
- **功能**: 反向查找UTF8子串
- **示例**:
```lua
local pos = utf8.inStrRev("中国人民万岁", "P", 10)
print(pos)  -- 输出: 4
```

### `utf8.left(s, len)`
- **功能**: 截取UTF8左边子串
- **示例**:
```lua
print(utf8.left("中国人民万岁", 2))  -- 输出: 中国
```

### `utf8.right(s, len)`
- **功能**: 截取UTF8右边子串
- **示例**:
```lua
print(utf8.right("中国人民万岁", 2))  -- 输出: 万岁
```

### `utf8.strCut(s, start, len)`
- **功能**: 移除UTF8子串
- **示例**:
```lua
print(utf8.strCut("中国人民万岁", 2, 2))  -- 输出: 中国万岁
```

### `utf8.strReverse(s)`
- **功能**: 反转UTF8字符串
- **示例**:
```lua
print(utf8.strReverse("中国人民万岁"))  -- 输出: 岁万民人国中
```
