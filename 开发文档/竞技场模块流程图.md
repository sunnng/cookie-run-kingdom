# 王国竞技场模块流程图（状态机版）

> 路径：`帅斌饼干/脚本/game/常规_王国竞技场/`
> 任务文件：`帅斌饼干/脚本/game/常规_王国竞技场/竞技场_任务.lua`
> 状态机：`core.state-machine`

---

## 1. 状态机全景

```mermaid
flowchart TD
    A[checkEnable<br/>检查开关与上限] -->|启用且未达上限| B[enter<br/>进入竞技场]
    A -->|未启用/已达上限| Z1[DONE]
    B -->|成功| C[syncStatus<br/>同步奖牌/门票/奖杯]
    B -->|失败| Z2[false 致命错误]
    C --> D[decideAction<br/>决策中枢]
    D -->|有门票| E[sweep<br/>扫描对手]
    D -->|无门票| F[autoBuy<br/>自动买票]
    D -->|升段标记| G[swipePageRight<br/>重置列表后 sweep]
    D -->|扫荡完成| H[freeRefresh<br/>处理刷新/倒计时]
    E -->|找到对手| J[battle<br/>执行战斗]
    E -->|门票不足| F
    E -->|达到上限| D
    J -->|成功| K[postBattle<br/>战后处理]
    J -->|失败| Z2
    K -->|升段| D
    K -->|继续扫荡| E
    F -->|买票成功| E
    F -->|未启用/达上限/买票失败| I
    H -->|免费刷新| C
    H -->|倒计时| I
    I --> Z1
```

---

## 2. 状态转换表

| 当前状态 | 触发条件 | 下一状态 | 说明 |
|---|---|---|---|
| `checkEnable` | `enabled == false` 或达 `maxBattles` | `DONE` | 直接结束 |
| `checkEnable` | 正常启用 | `enter` | 开始导航 |
| `enter` | 进入成功 | `syncStatus` | 同步资源 |
| `enter` | 进入失败 | `false` | 致命错误 |
| `syncStatus` | 同步完成 | `decideAction` | 进入决策 |
| `decideAction` | `promotionFlag == true` | `sweep` | 先右滑重置再扫荡 |
| `decideAction` | `sweepFinished == true` | `freeRefresh` | 统一处理刷新/倒计时 |
| `decideAction` | 门票 <= 0 | `autoBuy` | 尝试买票 |
| `decideAction` | 正常 | `sweep` | 继续扫荡 |
| `sweep` | 找到可战斗对手 | `battle` | 点击开战 |
| `sweep` | 门票不足 | `autoBuy` | 买票 |
| `sweep` | 达到上限 | `decideAction` | 设置 sweepFinished |
| `sweep` | 跳过/过滤后还有位置 | `KEEP` | 继续同状态 |
| `sweep` | 全部跳过/过滤完 | `decideAction` | 设置 sweepFinished |
| `battle` | 战斗完成 | `postBattle` | 战后处理 |
| `battle` | 战斗异常 | `false` | 致命错误 |
| `postBattle` | 判定升段 | `decideAction` | 设置 promotionFlag |
| `postBattle` | 还有位置 | `KEEP` → `sweep` | 继续扫荡 |
| `postBattle` | 全部完成 | `decideAction` | 设置 sweepFinished |
| `autoBuy` | 买票成功 | `sweep` | 重置扫荡位置 |
| `autoBuy` | 未启用/达上限/买票失败 | `leave` | 安全退出 |
| `freeRefresh` | 检测到免费刷新 | `syncStatus` | 点击刷新，开启新一轮 |
| `freeRefresh` | 未检测到免费刷新 | `leave` | 解析倒计时，设置下次进入时间 |
| `leave` | 离开完成 | `DONE` | 任务结束 |

## 3. 扫荡状态内部流程（sweep）

```mermaid
flowchart TD
    A[sweep 状态] --> B{达到 maxBattles?}
    B -->|是| C[设置 sweepFinished=true] --> D[decideAction]
    B -->|否| E{门票 > 0?}
    E -->|否| F[返回 autoBuy]
    E -->|是| G[syncStatus]
    G --> H[readOpponentInfo loc]
    H --> I{已战斗?}
    I -->|是| J[advanceSweep]
    I -->|否| K{奖杯差过滤?}
    K -->|是| J
    K -->|否| L[记录对手信息<br/>点击开战] --> M[返回 battle]
    J -->|还有位置| N[KEEP 继续 sweep]
    J -->|全部完成| O[设置 sweepFinished=true] --> D
```

---

## 4. 战后处理状态（postBattle）

```mermaid
flowchart TD
    A[postBattle 状态] --> B[更新胜/负/平统计]
    B --> C[门票 - 1]
    C --> D[syncStatus]
    D --> E{ensureLobby?}
    E -->|失败| Z1[false 致命错误]
    E -->|成功| F{opponentPage == 2?}
    F -->|是| G[swipePageLeft 复位]
    F -->|否| H[继续]
    G --> H
    H --> I[readOpponentInfo 原位置]
    I --> J{仍未显示已战?}
    J -->|是| K[设置 promotionFlag=true] --> L[decideAction]
    J -->|否| M[advanceSweep]
    M -->|还有位置| N[KEEP 继续 sweep]
    M -->|全部完成| O[设置 sweepFinished=true] --> L
```

---

## 5. 自动买票状态（autoBuy）

```mermaid
flowchart TD
    A[autoBuy 状态] --> B{autoBuyCount > 0?}
    B -->|否| I[leave]
    B -->|是| C{已买票数 >= 上限?}
    C -->|是| I
    C -->|否| D[buyTicket]
    D --> E[buyCount + 1]
    E --> F[syncStatus]
    F --> G{门票 > 0?}
    G -->|是| H[resetSweep<br/>返回 sweep]
    G -->|否| I[leave]
```

---

## 6. 免费刷新状态（freeRefresh）

```mermaid
flowchart TD
    A[freeRefresh 状态] --> B[waitFreeRefresh 最多10分钟]
    B -->|成功| C[resetSweep round+1]
    B -->|超时| C
    C --> D[返回 syncStatus]
```

## 7. 进入竞技场路由

```mermaid
flowchart TD
    A[Route.enter] --> B{已在大厅?}
    B -->|是| Z1[返回 true]
    B -->|否| C{在冒险页?}
    C -->|否| D{在王国主城?}
    D -->|否| Z2[返回 false]
    D -->|是| E[点击冒险按钮]
    E --> F[waitAdventure 30s]
    F -->|超时| Z2
    F -->|成功| G[OCR 点击王国竞技场]
    C -->|是| G
    G -->|失败| Z2
    G -->|成功| H[waitLobby 30s]
    H -->|超时| Z2
    H -->|成功| Z1
```

---

## 8. 单场战斗流程（runBattle）

```mermaid
flowchart TD
    A[runBattle] --> B[等待队伍选择页 30s]
    B -->|超时| Z1[返回 nil]
    B -->|成功| C[点击开始战斗]
    C --> D{弹出部署更多饼干?}
    D -->|是| E[点击确认]
    D -->|否| F
    E --> F{弹出未装载配料?}
    F -->|是| G[点击确认]
    F -->|否| H
    G --> H[等待队伍选择页消失]
    H -->|超时| Z1
    H -->|成功| I[等待结算页 120s]
    I -->|超时| Z1
    I -->|成功| J[OCR 读取战斗结果]
    J --> K[leaveSettlement]
    K -->|含升段跳过| M[返回 result]
```

---

## 9. 离开结算页 + 升段跳过流程

```mermaid
flowchart TD
    A[leaveSettlement] --> B{已在大厅?}
    B -->|是| Z1[返回 true]
    B -->|否| C{在结算页?}
    C -->|否| Z2[返回 false<br/>交给 skipPromotion]
    C -->|是| D[retryTap leaveRetry<br/>直到出现离开按钮]
    D -->|超时| Z3[返回 isLobby]
    D -->|成功| E[点击 leaveTap]
    E --> F[sleep 2000]
    F --> G[retryTap promotionSkip<br/>直到出现大厅特征]
    G -->|超时| Z3
    G -->|成功| H[sleep 1500]
    H --> I{仍在大厅?}
    I -->|是| Z1
    I -->|否| J[retryTap promotionSkip<br/>直到再次出现大厅特征]
    J -->|成功| Z1
    J -->|超时| Z3
```
