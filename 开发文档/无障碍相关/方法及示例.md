# 无障碍方法文档

## 手势操作

### `Gesture:new()`
- **功能**: 创建新手势对象
- **示例**:
```lua
local gesture = Gesture:new()
local path = Path:new()
path:addPoint(100,500)
path:addPoint(500,500)
gesture:addPath(path)
gesture:dispatch()
```

### `Path:addPoint(x,y)`
- **功能**: 添加手势路径点
- **参数**:
  - `x,y`: 屏幕坐标
- **注意**: 需配合`setDurTime`设置滑动时间

## 节点操作

### `nodeLib.click(node)`
- **功能**: 点击无障碍节点
- **示例**:
```lua
local node = nodeLib.findOne({class="Button"})
if node then
  nodeLib.click(node)
end
```

### `nodeLib.setText(node,text)`
- **功能**: 设置可编辑节点文本
- **示例**:
```lua
local edit = nodeLib.findOne({class="EditText"})
if edit then
  nodeLib.setText(edit,"测试文本")
end
```

## 节点查找

### `nodeLib.findAll(selector)`
- **功能**: 查找所有匹配节点
- **选择器示例**:
```lua
{
  class="android.widget.Button",
  text="确定",
  package="com.example.app"
}
```

### `nodeLib.findParentNode(node)`
- **功能**: 查找父节点
- **用途**: 定位节点层级结构

## 服务控制

### `nodeLib.isAccServiceOk()`
- **功能**: 检查无障碍服务状态
- **返回值**: 布尔值

### `nodeLib.openSnapService()`
- **功能**: 开启截图服务
- **注意**: 需用户手动确认权限

local ret = nodeLib.findByIndex("0|1|2")

	nodeLib.click(ret)
end

方法名称:nodeLib.findAll 查找所有满足要求的节点
语法:local node = nodeLib.findAll(selector,fuzzy_match)
参数:selector 是一个表格,里面的键值对通过节点工具勾选直接生成
fuzzy_match:是布尔类型，true表示使用正则匹配去查询，false表示完全匹配
返回值:查找所有满足要求的节点
local ret = nodeLib.findAll({class="android.widget.TextView",id="",package="com.nx.nxproj",text="名称: tmp.lr"},false)
if ret ~= nil then
	print(ret)
end

方法名称:nodeLib.findChildNodes 查找所有满足要求的节点的所有子节点

end

方法名称:nodeLib.findOne 查找出一个节点
语法:local node = nodeLib.findInNode(src,dst,isfindall,fuzzy_match)
参数:src是要被查找的节点对象,dst是要查找的节点对象
isfindall:布尔类型，是否查找并返回所有符合要求的节点，true返回所有，false找到一个就返回
fuzzy_match:是布尔类型，true表示使用正则匹配去查询，false表示完全匹配
返回值:返回一个满足要求的节点或者节点数组
local ret = nodeLib.findOne(0,0,0,0,{class="android.widget.FrameLayout",id="com.nx.nxproj.assist:id/title"},true)
if ret ~= nil then
	local r = nodeLib.findInNode(ret,{text=".*新建.*"},true,true)
	print(ret)
	if r~= nil then
		for i=1,#r do
			print(r[i])
		end
	end
end

方法名称:nodeLib.isSnapServiceOk 检测截图服务是否开启(如果是root或者激活模式不需要用这个去检测，因为使用的截图方法不一样)

返回值:布尔类型
local r = nodeLib.isAccServiceOk()
if r then
	print("截图服务已经开启")
else
	print("截图服务没有开启")
end

方法名称:nodeLib.keepNode 锁定当前界面到内存，下次节点查找查找都会直接从内存中获取该节点
语法:local node = nodeLib.keepNode()
注意:这个方法是配合旧版节点函数使用注意和lockNode(新版的节点函数)区别开了
返回值:成功返回真失败返回假
nodeLib.keepNode()
local search_edit = {class="android.widget.EditText",id="com.android.launcher3:id/search_edit"}
local input_edit = {class="android.widget.EditText",id="com.android.launcher3:id/input_edit"}
local ret1 = nodeLib.findOne(search_edit,true)
local ret2 = nodeLib.findOne(input_edit,true)
if ret1 ~= nil and ret2 ~= nil then
	nodeLib.setText(input_edit,"abc")
end
nodeLib.releaseNode()

方法名称:nodeLib.openSnapService 打开截图服务(这个方法一般用于手动模式，调用这方法会弹出允许截图窗口需要手动完成点击确认)
语法:local r = nodeLib.openSnapService()
返回值:布尔类型
local r = nodeLib.openSnapService()
if r then
	if nodeLib.isAccServiceOk() == false then
		r = false
	end
end
if r then
	print("截图服务开启成功")
else
	print("截图服务开启失败")
end

方法名称:nodeLib.setNodeMode 设置获取节点模式
语法:local succ = nodeLib.setNodeMode(mode,enableGetInVisibleNode)
参数:mode整数型,为0表示获取所节点，为1表示获取节点模式为简单模式(不重要的节点将被过滤),enableGetInVisibleNode表示抓取节点的时候是否获取不可见的节点
返回值:布尔类型
nodeLib.setNodeMode(1,false)
