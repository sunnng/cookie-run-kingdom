# JSON方法文档

## 基础转换

### `jsonLib.decode(json_str)`
- **功能**: JSON字符串转Lua表
- **示例**:
```lua
local json = '{"name":"John", "age":30}'
local data = jsonLib.decode(json)
print(data.name) -- 输出: John
```

### `jsonLib.encode(lua_table)`
- **功能**: Lua表转JSON字符串
- **示例**:
```lua
local data = {name="John", age=30}
local json = jsonLib.encode(data)
print(json) -- 输出: {"name":"John","age":30}
```

## 高级用法

### 嵌套结构处理
```lua
-- 复杂结构转换示例
local complex = {
  users = {
    {name="Alice", age=25},
    {name="Bob", age=30}
  },
  timestamp = os.time()
}

local json = jsonLib.encode(complex)
print(json)

-- 解码回Lua表
local decoded = jsonLib.decode(json)
print(decoded.users[1].name) -- 输出: Alice
```

### 错误处理
```lua
local ok, result = pcall(jsonLib.decode, 'invalid json')
if not ok then
  print("JSON解析错误:", result)
end
