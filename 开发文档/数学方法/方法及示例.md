# 数学方法文档

## 基础运算

### `math.abs(x)`
- **功能**: 计算绝对值
- **示例**:
```lua
print(math.abs(-3.5))  -- 输出: 3.5
```

### `math.floor(x)`/`math.ceil(x)`
- **功能**: 向下/向上取整
- **示例**:
```lua
print(math.floor(3.7))  -- 输出: 3
print(math.ceil(3.2))   -- 输出: 4
```

## 三角函数

### `math.sin(x)`/`math.cos(x)`/`math.tan(x)`
- **注意**: 参数需为弧度
- **示例**:
```lua
local radians = math.rad(30)  -- 30度转弧度
print(math.sin(radians))     -- 输出: 0.5
```

### `math.rad(x)`/`math.deg(x)`
- **功能**: 角度弧度互转
- **示例**:
```lua
print(math.deg(math.pi))  -- 输出: 180
```

## 高级运算

### `math.random([m[,n]])`
- **注意**: 需先设置随机种子
- **示例**:
```lua
math.randomseed(os.time())
print(math.random(1,100))  -- 1-100随机整数
```

### `math.modf(x)`
- **功能**: 分离整数和小数部分
- **示例**:
```lua
local int, frac = math.modf(3.14)
print(int, frac)  -- 输出: 3   0.14
```

## 类型检查

### `math.type(x)`
- **返回值**: "integer", "float" 或 nil
- **示例**:
```lua
print(math.type(5))    -- 输出: integer
print(math.type(5.0))  -- 输出: float
```

## 其他方法

### `math.acos(x)`
- **功能**: 反余弦
- **示例**:
```lua
print(math.acos(0.5))  -- 输出: 1.0471975511966
```

### `math.asin(x)`
- **功能**: 反正弦
- **示例**:
```lua
print(math.asin(0.5))  -- 输出: 0.5235987755983
```

### `math.atan(x)`
- **功能**: 反正切
- **示例**:
```lua
print(math.atan(1))  -- 输出: 0.78539816339745
```

### `math.exp(x)`
- **功能**: e的x次方
- **示例**:
```lua
print(math.exp(2))  -- 输出: 7.3890560989307
```

### `math.log(x)`
- **功能**: 自然对数
- **示例**:
```lua
print(math.log(2.0))  -- 输出: 0.69314718055995
```

### `math.max(x, ...)`
- **功能**: 最大值
- **示例**:
```lua
print(math.max(1, 5, 3, 6, 7, 8))  -- 输出: 8
```

### `math.min(x, ...)`
- **功能**: 最小值
- **示例**:
```lua
print(math.min(1, 5, 3, 6, 7, 8))  -- 输出: 1
```

### `math.sqrt(x)`
- **功能**: 平方根
- **示例**:
```lua
print(math.sqrt(2.0))  -- 输出: 1.4142135623731
```

### `math.tointeger(x)`
- **功能**: 转换为整数
- **示例**:
```lua
print(math.tointeger("666"))  -- 输出: 666
```

### `math.ult(m, n)`
- **功能**: 无符号整数比较
- **示例**:
```lua
print(math.ult(6, 8))  -- 输出: true
```
