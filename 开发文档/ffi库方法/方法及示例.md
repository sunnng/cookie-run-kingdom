# FFI库方法文档

## 基础功能

### `ffi.cdef(str)`
- **功能**: 声明C语言类型和函数
- **示例**:
```lua
ffi.cdef[[
  int printf(const char *fmt, ...);
  int getpid(void);
]]
```

### `ffi.load(name)`
- **功能**: 加载动态链接库
- **示例**:
```lua
local libc = ffi.load("libc.so.6")
print(libc.getpid())
```

## 类型操作

### `ffi.new(ctype)`
- **功能**: 创建C类型变量
- **示例**:
```lua
ffi.cdef[[ typedef struct { int x; } point_t; ]]
local pt = ffi.new("point_t")
pt.x = 10
```

### `ffi.cast(ctype, value)`
- **功能**: 类型转换
- **示例**:
```lua
local num = ffi.cast("int", 3.14)
```

## 内存操作

### `ffi.copy(dst, src, len)`
- **功能**: 内存拷贝
- **示例**:
```lua
local buf = ffi.new("char[?]", 10)
ffi.copy(buf, "hello", 5)
```

### `ffi.string(ptr, len)`
- **功能**: 指针转字符串
- **示例**:
```lua
local str = ffi.string(buf, 5)
print(str)  -- 输出: hello
```
