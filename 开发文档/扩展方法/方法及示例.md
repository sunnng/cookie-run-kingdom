# 扩展方法文档

## 基础功能

### `MD5(str)`
- **功能**: 计算字符串MD5值
- **参数**: `str` 要计算的字符串
- **示例**:
```lua
print(MD5("hello"))  -- 输出: 5d41402abc4b2a76b9719d911017c592
```

### `encodeBase64(str)`/`decodeBase64(str)`
- **功能**: Base64编解码
- **示例**:
```lua
local encoded = encodeBase64("test")
local decoded = decodeBase64(encoded)
```

## 文件操作

### `readFile(path)`/`writeFile(path,str)`
- **功能**: 文件读写
- **参数**:
  - `path`: 文件路径
  - `str`: 要写入的内容
- **示例**:
```lua
writeFile("/sdcard/test.txt", "data")
print(readFile("/sdcard/test.txt"))
```

## 图像处理

### `getImage(path)`
- **功能**: 读取图像像素数据
- **返回值**: 宽度,高度,像素数组

### `qrDecode(path)`
- **功能**: 解析二维码
- **返回值**: 状态码(1成功),解析结果
- **示例**:
```lua
local ok, text = qrDecode("/sdcard/qrcode.png")
if ok == 1 then print(text) end
```

## 多线程

### `beginThread(func)`
- **功能**: 启动新线程
- **注意**: 最多10个并发线程
- **示例**:
```lua

  while true do
    print("Running in thread")
    sleep(1000)
  end
end)
```

### `LuaEngine.loadApk(name)`
- **功能**: 加载apk插件
- **参数**: `name` apk名称或绝对路径
- **返回值**: 插件对象
- **示例**:
```lua
local loader = LuaEngine.loadApk("app-release.apk")
printEx(loader)
if loader ~= nil then
  local plugin = loader.loadClass("com.lrapp.plugin.Plugin")
  if plugin ~= nil then
    local c = plugin.add(1,2)
    print(c)
  end
end
```

### `bdOcr(url,apikey,secretkey,[type])`
- **功能**: 百度OCR识别
- **参数**:
  - `url`: 图片地址
  - `apikey`: 百度API Key
  - `secretkey`: 百度Secret Key
  - `[type]`: 识别类型（0-3）
- **返回值**: JSON结果
- **示例**:
```lua
local path = "/mnt/sdcard/test.png"
snapShot(path,66,119,248,186)
local ret = bdOcr(path,"Yub8co6xdEfmMHykqg38GXhB","Hyhbsc1yGf78E65ETcn1wZvPV2lnKXwZ")
print("bdocr:",ret)
```

### `binaryImage(srcimage,dstimage,[ThresholdTypes],[thresh],[maxthresh])`
- **功能**: 图像二值化
- **参数**:
  - `srcimage`: 源图像
  - `dstimage`: 目标图像
  - `[ThresholdTypes]`: 阈值类型
  - `[thresh]`: 阈值
  - `[maxthresh]`: 最大阈值
- **返回值**: 成功标志（1成功）
- **示例**:
```lua
local path = "/mnt/sdcard/test.png"
local save = "/mnt/sdcard/save.png"
snapShot(path,200,211,350,330,8,150,255)
local ret = binaryImage(path,save)
if ret == 1 then
  local id = createHUD()
  showHUD(id,"HelloWorld!",12,"0xffff0000",save,0,100,100,200,200)
else
  print("binaryimage failed")
end
sleep(100000)
```

### `mysql.closeSQL(handle)`
- **功能**: 关闭MySQL连接
- **参数**: `handle` 连接句柄
- **返回值**: 成功标志（true成功）
- **示例**:
```lua
require("mysql")
local con,err = mysql.connectSQL("192.168.1.24",3306,"mydatabase","root","mtest@123",20000)
if con ~= nil then
  mysql.closeSQL(con)
else
  print("连接失败:"..err)
end
```

### `mysql.connectSQL(host,port,database,user,password,timeout)`
- **功能**: 连接MySQL数据库
- **参数**:
  - `host`: 主机地址
  - `port`: 端口号
  - `database`: 数据库名称
  - `user`: 用户名
  - `password`: 密码
  - `timeout`: 超时时间（毫秒）
- **返回值**: 连接句柄，错误信息
- **示例**:
```lua
require("mysql")
local con,err = mysql.connectSQL("192.168.1.24",3306,"mydatabase","root","mtest@123",20000)
if con ~= nil then
  local sql = string.format("insert persons(name,height,age) values('%s',%d,%d)","Limi",197,23)
  local r,err = mysql.executeSQL(con,sql)
  if err ~= nil then
    print("执行错误:"..err)
  end
  local rs,err = mysql.executeQuerySQL(con,"select * from persons")
  if err ~= nil then
    print("执行错误:"..err)
  end
  print(rs,err)
  mysql.closeSQL(con)
else
  print("连接失败:"..err)
end
```
