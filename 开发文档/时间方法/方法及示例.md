# 时间方法文档

## 时间获取

### `getNetWorkTime()`
- **功能**: 获取网络标准时间
- **返回值**: 格式为`年-月-日_时-分-秒`的字符串
- **示例**:
```lua
print("当前网络时间:", getNetWorkTime())
```

### `os.time([table])`
- **功能**: 获取时间戳(秒)
- **参数**: 可选table参数
- **示例**:
```lua
-- 获取当前时间戳
local timestamp = os.time()
-- 获取指定日期时间戳
local date = {year=2023, month=11, day=15}
local dateStamp = os.time(date)
```

## 时间格式化

### `os.date(format[, time])`
- **功能**: 格式化时间戳
- **常用格式**:
  - `%Y-%m-%d %H:%M:%S`: 标准格式
  - `%x`: 日期(09/16/98)
  - `%X`: 时间(23:48:10)
- **示例**:
```lua
-- 当前时间格式化
local now = os.date("%Y-%m-%d %H:%M:%S")
-- 时间戳转日期
local dateStr = os.date("%Y年%m月%d日", 1630000000)
```

## 计时功能

### `tickCount()`
- **功能**: 获取脚本运行时长(毫秒)
- **用途**: 性能测试
- **示例**:
```lua
local start = tickCount()
-- 执行一些操作
print("耗时:", tickCount()-start, "ms")
```

## 其他时间方法

### `systemTime()`
- **功能**: 获取系统时间戳(毫秒)
- **示例**:
```lua
local sysTime = systemTime()
print("系统时间戳:", sysTime)
```

### `os.date("*t", time)`
- **功能**: 获取时间戳的详细信息
- **示例**:
```lua
local time = os.time()
local detail = os.date("*t", time)
print("小时:", detail.hour)
```

### 获取7天前的时间
```lua
local now = os.date("%Y-%m-%d %H:%M:%S")
local year = string.sub(now,1,4)
local month = string.sub(now,6,7)
local day = string.sub(now,9,10)
local hour = string.sub(now,12,13)
local minute = string.sub(now,15,16)
local second = string.sub(now,18,19)
local ta = {
    year=year,
    month=month,
    day=day-7,
    hour=hour,
    min=minute,
    sec=second
}
local t = os.date("%Y-%m-%d %H:%M:%S", os.time(ta))
print("7天前时间:", t)
```

### 获取本年2月份的天数
```lua
local now = os.date("%Y-%m-%d %H:%M:%S")
local year = string.sub(now,1,4)
local month = string.sub(now,6,7)
local day = string.sub(now,9,10)
local hour = string.sub(now,12,13)
local minute = string.sub(now,15,16)
local second = string.sub(now,18,19)
local ta = {
    year=year,
    month=3,
    day=0,
}
local days = os.date("%d", os.time(ta))
print("本年2月份天数:", days)
