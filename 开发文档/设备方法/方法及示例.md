# 设备方法文档

## 设备信息

### `getBrand()`/`getModel()`
- **功能**: 获取设备品牌和型号
- **示例**:
```lua
print("设备:", getBrand(), getModel())
```

### `getDisplaySize()`
- **功能**: 获取屏幕分辨率
- **返回值**: 宽度,高度
- **示例**:
```lua
local w, h = getDisplaySize()
print("分辨率:", w, "x", h)
```

## 应用控制

### `runApp(package)`/`stopApp(package)`
- **功能**: 启动/停止应用
- **参数**: 应用包名
- **示例**:
```lua
runApp("com.tencent.mm")  -- 启动微信
sleep(5000)
stopApp("com.tencent.mm")  -- 停止微信
```

## 系统控制

### `setWifiEnable(state)`
- **功能**: 开关WiFi
- **参数**: true开启, false关闭

### `vibrate(duration)`
- **功能**: 设备震动
- **参数**: 震动时长(毫秒)
- **示例**:
```lua
vibrate(500)  -- 震动0.5秒
```

## 脚本控制

### `exitScript()`/`restartScript()`
- **功能**: 退出/重启脚本
- **示例**:
```lua
setStopCallBack(function()
  print("脚本即将退出")
end)
exitScript()
```

## 文件操作

### `getSdPath()`
- **功能**: 获取SD卡路径
- **用途**: 文件操作基准路径

### `extractAssets(src, dest)`
- **功能**: 释放资源文件
- **参数**:
  - src: 资源路径
  - dest: 目标路径

## 高级功能

### `exec(cmd)`
- **功能**: 执行系统命令
- **示例**:
```lua
print(exec("ls /sdcard"))
```

### `setDpiToVir(dpi)`
- **功能**: 设置虚拟DPI(需root)
- **注意**: 需配套使用`setDpiToRealy()`恢复

import("android.widget.Toast")
import('java.lang.*')
import('java.util.*')
import('android.widget.*')
import('android.os.Build')

local ctx = LuaEngine.getContext()

nodeLib.openAccessibility()

function onNotifyEvent(e, arg)
	print("系统消息:", e, arg)
	return "1"
end

setNotifyEventCallBack(onNotifyEvent)

ffi.cdef [[
	int getuid(void);
]]

local clib = ffi.load("libc")


function showToast(i)
	local uid = clib.getuid()
	if uid ~= 0 and uid ~= 2000 then
		LuaEngine.runOnUIThread(Runnable {
			run = function()
				local toast = Toast.makeText(ctx, "hello" .. i, Toast.LENGTH_SHORT)
				toast.show()
			end
		})
	else
		print("当前环境无法测试android原生的toast请将运行才最高权限中关闭")
	end
end

for i = 1, 100000, 1 do
	showToast(i)
	sleep(5000)
end

方法名称: 保护调用并捕获错误
语法: xpcall(func, errHandler)
参数说明:
  func - 要执行的主函数
  errHandler - 错误处理函数（接收错误信息参数）
功能说明: 
  1. 提供保护模式调用函数，捕获执行过程中的错误
  2. 当主函数(func)发生错误时，自动调用错误处理函数(errHandler)
  3. 错误处理函数接收错误信息(err)作为参数
  4. 返回两个值: [是否成功, 返回值/错误信息]
  5. 相比pcall，xpcall允许自定义错误处理逻辑
典型用途:
  异常处理、崩溃日志记录、程序安全执行
-- 示例 1: 基础错误捕获
function main()
  error("数据加载失败")
end

xpcall(main, function(err)
  print("错误信息:", err)
  print("堆栈跟踪:", debug.traceback())
end)

-- 示例 2: 带返回值的保护调用
function calculate()
  return 10/0 -- 触发除零错误
end

local success, result = xpcall(calculate, function(err)
  print("计算错误:", err)
  return "N/A" -- 返回替代值
end)

print("计算结果:", success and result or "失败")

-- 示例 3: 嵌套错误处理
function task1()
  error("任务1失败")
end

function errorHandler(err)
  print("顶级错误处理:", err)
  print(debug.traceback("错误追踪:", 2))
end

xpcall(task1, errorHandler)
