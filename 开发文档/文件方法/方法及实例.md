# 文件方法文档

## 文件基础操作

### `io.open(path, mode)`
- **功能**: 打开文件
- **模式参数**:
  - `"r"`: 只读(默认)
  - `"w"`: 只写(覆盖)
  - `"a"`: 追加
  - `"r+"`: 读写(保留内容)
  - `"w+"`: 读写(清空内容)
- **示例**:
```lua
local file = io.open("/sdcard/test.txt", "r")
if file then
  print(file:read("*a"))
  file:close()
end
```

### `io.lines([filename])`
- **功能**: 文件行迭代器
- **示例**:
```lua
for line in io.lines("/sdcard/log.txt") do
  print(line)
end
```

## 文件读写

### `file:read(format)`
- **读取格式**:
  - `"*a"`: 读取全部内容
  - `"*l"`: 读取一行(默认)
  - `"*n"`: 读取数字
- **示例**:
```lua
local content = file:read("*a")  -- 读取整个文件
```

### `file:write(...)`
- **功能**: 写入内容
- **示例**:
```lua
local file = io.open("/sdcard/data.txt", "w")
file:write("Line 1\n", "Line 2\n")
file:close()
```

## 文件控制

### `file:seek([whence][, offset])`
- **定位参数**:
  - `"set"`: 文件开头(默认)
  - `"cur"`: 当前位置
  - `"end"`: 文件末尾
- **示例**:
```lua
file:seek("end", -10)  -- 定位到文件末尾前10字节
```

### `file:setvbuf(mode[, size])`
- **缓冲模式**:
  - `"no"`: 无缓冲
  - `"full"`: 全缓冲
  - `"line"`: 行缓冲
- **示例**:
```lua
file:setvbuf("line")  -- 设置行缓冲
```

## 文件输入输出

### `io.input([file])`
- **功能**: 设置默认输入文件
- **示例**:
```lua
local handle = io.input("/mnt/sdcard/test.txt")
print(io.read("*a"))
io.close(handle)
```

### `io.output([file])`
- **功能**: 设置默认输出文件
- **示例**:
```lua
local handle = io.output("/mnt/sdcard/test.txt")
io.write("123")
io.close(handle)
```

### `io.read()`
- **功能**: 读取文件内容
- **示例**:
```lua
io.input("/mnt/sdcard/test.txt")
while true do
  local ReadContent = io.read()
  if ReadContent == nil then
    break
  end
  print(ReadContent)
end
io.close()
```

### `io.write(...)`
- **功能**: 写入文件内容
- **示例**:
```lua
local handle = io.output("/mnt/sdcard/test.txt")
io.write(tostring("123"))
io.close(handle)
```
