# 数组方法文档

## 基本操作

### `#array`
- **功能**: 获取数组长度
- **示例**:
```lua
local arr = {1,2,3}
print(#arr)  -- 输出: 3
```

### `table.concat(table [, sep])`
- **功能**: 连接数组元素为字符串
- **参数**:
  - `sep`: 分隔符(可选)
- **示例**:
```lua
local arr = {"a","b","c"}
print(table.concat(arr, ","))  -- 输出: a,b,c
```

## 元素操作

### `table.insert(table, [pos,] value)`
- **功能**: 插入元素到数组
- **示例**:
```lua
local arr = {1,2}
table.insert(arr, 3)      -- 末尾插入
print(arr[3])            -- 输出: 3
table.insert(arr, 1, 0)  -- 开头插入
print(arr[1])            -- 输出: 0
```

### `table.remove(table [, pos])`
- **功能**: 移除数组元素
- **示例**:
```lua
local arr = {1,2,3}
table.remove(arr)  -- 移除最后一个
print(#arr)        -- 输出: 2
```

## 高级操作

### `table.sort(table [, comp])`
- **功能**: 数组排序
- **示例**:
```lua
local arr = {3,1,2}
table.sort(arr)
print(table.concat(arr))  -- 输出: 123
```

### `ipairs(table)`
- **功能**: 数组迭代器
- **示例**:
```lua
for i,v in ipairs({"a","b"}) do
  print(i,v)  -- 输出: 1 a    2 b
end
```
