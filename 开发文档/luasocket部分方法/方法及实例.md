# LuaSocket方法文档

## HTTP请求

### `http.request(options)`
- **功能**: 发起HTTP/HTTPS请求
- **参数说明**:
```lua
{
  url = "http://example.com",  -- 请求URL
  method = "GET",           -- 请求方法
  headers = {                -- 请求头
    ["Content-Type"] = "application/json"
  },
  timeout = 10               -- 超时时间(秒)
}
```
- **示例**:
```lua
local res = http.request{
  url = "https://api.example.com/data",
  method = "GET"
}
if res.statuscode == 200 then
  print(res.content)
end
```

## HTTPS请求

### `https.request()`
- **功能**: 安全的HTTPS请求
- **示例**:
```lua
local https = require("ssl.https")
local body = {}
https.request{
  url = "https://example.com",
  sink = ltn12.sink.table(body)
}
print(table.concat(body))
```

## 响应处理

### `ltn12.sink.table()`
- **功能**: 将响应内容收集到表中
- **示例**:
```lua
local body = {}
http.request{
  url = "http://example.com",
  sink = ltn12.sink.table(body)
}
-- 打印完整响应内容
print(table.concat(body))
```

## 代码示例

```lua
-- 设置 HTTPS 请求的 URL
local url = "http://www.baidu.com/"

-- 创建一个用于保存网页内容的变量
local responseBody = {}

-- 发起 HTTPS 请求
local _, status, _, _ = https.request{
  url = url,
  method = "GET",
  protocol = "tlsv1_2",
  sink = ltn12.sink.table(responseBody)
}

-- 检查请求是否成功
if status == 200 then
  print(responseBody)
else
  print("Request failed:", status)
end
```
