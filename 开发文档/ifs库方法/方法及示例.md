# IFS库方法文档

## 目录操作

### `lfs.attributes(path)`
- **功能**: 获取文件/目录属性
- **返回值**: 包含文件大小、修改时间等信息的表
- **示例**:
```lua
local attrs = lfs.attributes("/sdcard/test.txt")
print("Size:", attrs.size)
print("Modification time:", attrs.modification)
```

### `lfs.dir(path)`
- **功能**: 遍历目录内容
- **示例**:
```lua
for file in lfs.dir("/sdcard") do
  if file ~= "." and file ~= ".." then
    print(file)
  end
end
```

## 文件锁控制

### `lfs.lock(file, mode)`
- **功能**: 文件锁定
- **模式**: "r"(读锁), "w"(写锁)
- **示例**:
```lua
local file = io.open("test.txt", "w")
if lfs.lock(file, "w") then
  -- 安全写入操作
  file:write("data")
  lfs.unlock(file)
end
file:close()
```

## 目录管理

### `lfs.mkdir(path)`
- **功能**: 创建目录
- **示例**:
```lua
if lfs.mkdir("/sdcard/new_dir") then
  print("Directory created")
end
```

### `lfs.rmdir(path)`
- **功能**: 删除空目录
- **示例**:
```lua
if lfs.rmdir("/sdcard/empty_dir") then
  print("Directory removed")
end
```

## 高级功能

### `lfs.touch(path)`
- **功能**: 更新文件时间戳
- **示例**:
```lua
lfs.touch("old_file.txt")  -- 更新访问和修改时间
```

### `lfs.symlinkattributes(path)`
- **功能**: 获取符号链接属性
- **示例**:
```lua
local attrs = lfs.symlinkattributes("symlink")
print("Target:", attrs.target)
```

## 其他方法

### `lfs.chdir(path)`
- **功能**: 切换当前工作目录
- **示例**:
```lua
lfs.chdir("/sdcard")
```

### `lfs.currentdir()`
- **功能**: 获取当前工作目录
- **示例**:
```lua
local currentDir = lfs.currentdir()
print(currentDir)
```

### `lfs.link(src, dest)`
- **功能**: 创建硬链接
- **示例**:
```lua
lfs.link("src_file.txt", "dest_file.txt")
```

### `lfs.lock_dir(path)`
- **功能**: 锁定目录
- **示例**:
```lua
lfs.lock_dir("/sdcard/locked_dir")
```

### `lfs.unlock(file)`
- **功能**: 解锁文件
- **示例**:
```lua
local file = io.open("test.txt", "w")
lfs.lock(file, "w")
-- ...
lfs.unlock(file)
file:close()
