# 网络方法文档

## HTTP请求

### `httpGet(url)`
- **功能**: 同步GET请求
- **示例**:
```lua
local response, code = httpGet("https://api.example.com/data")
print("状态码:", code, "响应:", response)
```

### `asynHttpPost(callback,url,postdata)`
- **功能**: 异步POST请求
- **回调参数**: response, status_code
- **示例**:
```lua
function callback(res, code)
  print("异步响应:", res)
end
asynHttpPost(callback, "https://api.example.com", "data=test")
```

## WebSocket通信

### `startWebSocket(url,callbacks)`
- **功能**: 建立WebSocket连接
- **回调方法**:
  - `onOpened(handle)`
  - `onClosed(handle)`
  - `onError(handle)`
  - `onRecv(handle,message)`
- **示例**:
```lua
local wsHandle = startWebSocket("ws://echo.websocket.org", {
  onOpened = function(h) print("连接成功") end,
  onRecv = function(h,msg) print("收到:",msg) end
})
```

## 文件传输

### `downloadFile(url,savepath)`
- **功能**: 下载文件
- **进度回调**:
```lua
downloadFile("http://example.com/file.zip", "/sdcard/file.zip", 
  function(progress)
    print("下载进度:", progress.."%")
  end)
```

### `uploadFile(url,filepath)`
- **功能**: 上传文件
- **示例**:
```lua
local result = uploadFile("https://api.example.com/upload", "/sdcard/test.jpg")
print("上传结果:", result)
```

## 邮件功能

### `LuaEngine.sendMail()`
- **功能**: 发送邮件
- **完整示例**:
```lua
LuaEngine.sendMail("user@example.com", "password", 
  "recipient@mail.com", "smtp.example.com", true,
  "邮件标题", "邮件内容", {
    onSuccess = function() print("发送成功") end,
    onFailed = function(err) print("失败:",err) end
  })
```

参数说明:

	sleep(1000)

方法名称:打开一个websocket连接
语法:startWebSocket(url,onOpened,onClosed,onError,onRecv)
参数说明:
url:ws地址,目前只支持ws协议wss暂不支持
onOpend:当连接服务器成功后会回调这个函数
onClosed:当连接服务器断开后会回调这个函数
onError:当连接服务器失败后会回调这个函数
onRecv:当有接收到数据时会回调这个函数
使用事项:以上所有回调函数是在单独的一个线程里面，请不要在回调函数中阻塞运行
或者调用停止(exitScript)或者重启脚本(restartScript)函数，如果非要使用请在setTimer回调函数中执行，
因为setTimer回调函数是运行在主线程中
返回值:返回一个数字类型，表示这个websocket链接的句柄
local ws = nil

function wLog(text)

end

function onOpened(handle)
