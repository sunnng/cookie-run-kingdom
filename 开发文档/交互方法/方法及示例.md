# 交互方法文档

## 窗口管理

### `showUI(uifile,[w],[h],[onUIEvent])`
- **功能**: 显示自定义UI界面
- **参数**:
  - `uifile`: UI文件路径
  - `w,h`: 窗口尺寸(-1表示填满)
  - `onUIEvent`: 事件回调函数
- **返回值**: 配置JSON字符串
- **示例**:
```lua
function onUIEvent(handle,event,arg1,arg2,arg3)
  if event == "onclick" then
    print("点击控件:", arg2)
  end
end
showUI("test.ui", 600, 400, onUIEvent)
```

### `closeWindow(handle,saveConfig)`
- **功能**: 关闭窗口
- **参数**:
  - `handle`: 窗口句柄
  - `saveConfig`: 是否保存配置

## HUD显示

### `createHUD()`
- **功能**: 创建HUD实例
- **返回值**: HUD句柄

### `showHUD(id,text,size,color,bg,pos,x,y,width,height)`
- **功能**: 显示/更新HUD内容
- **参数**:
  - `text`: 显示文本
  - `color`: 文本颜色(ARGB格式)
  - `bg`: 背景(颜色或图片路径)
  - `pos`: 定位模式(0-左上角,1-居中)

## 拖拽视图

### `createDragView(x,y,width,height,imagePath)`
- **功能**: 创建可拖动视图
- **示例**:
```lua
local handle = createDragView(100,100,50,50,"0xff00ff00")
setDragViewOnClick(handle, function(h)
  print("视图被点击")
  removeDragView(h)
end)
```

## 消息提示

### `toast(text,[x,y,textsize])`
- **功能**: 显示临时消息
- **参数**:
  - `text`: 消息内容
  - `x,y`: 显示位置(0,0为自动居中)
- **示例**:
```lua
toast("操作成功", 0, 0, 14)  -- 居中显示
```

## 控件管理

### `setUIBackground(handle,pageid,idname,bg)`
- **功能**: 设置控件背景
- **参数**:
  - `handle`: 窗口句柄
  - `pageid`: 控件所在页面索引
  - `idname`: 控件ID
  - `bg`: 背景颜色或图片路径

### `setUICheck(handle,pageidx,idname,selected)`
- **功能**: 设置多选框选中状态
- **参数**:
  - `handle`: 窗口句柄
  - `pageidx`: 控件所在页面索引
  - `idname`: 控件ID
  - `selected`: 是否选中

### `setUIEnable(handle,pageidx,idname,enable)`
- **功能**: 设置控件是否可用
- **参数**:
  - `handle`: 窗口句柄
  - `pageidx`: 控件所在页面索引
  - `idname`: 控件ID
  - `enable`: 是否可用

### `setUISelect(handle,pageidx,idname,index)`
- **功能**: 设置单选框或下拉框选中项
- **参数**:
  - `handle`: 窗口句柄
  - `pageidx`: 控件所在页面索引
  - `idname`: 控件ID
  - `index`: 选中项索引

### `setUIText(handle,pageidx,idname,text)`
- **功能**: 设置控件文本
- **参数**:
  - `handle`: 窗口句柄
  - `pageidx`: 控件所在页面索引
  - `idname`: 控件ID
  - `text`: 文本内容

### `setUITextColor(handle,pageidx,idname,textcolor)`
- **功能**: 设置控件文本颜色
- **参数**:
  - `handle`: 窗口句柄
  - `pageidx`: 控件所在页面索引
  - `idname`: 控件ID
  - `textcolor`: 文本颜色(ARGB格式)

### `setUIVisible(handle,pageidx,idname,visible)`
- **功能**: 设置控件可见性
- **参数**:
  - `handle`: 窗口句柄
  - `pageidx`: 控件所在页面索引
  - `idname`: 控件ID
  - `visible`: 可见性(0-可见,8-隐藏)

## 浏览器管理

### `setUIWebViewUrl(handle,pageidx,idname,url)`
- **功能**: 设置浏览器地址
- **参数**:
  - `handle`: 窗口句柄
  - `pageidx`: 控件所在页面索引
  - `idname`: 控件ID
  - `url`: 浏览器地址

## HUD管理

### `hideHUD(id)`
- **功能**: 隐藏HUD
- **参数**:
  - `id`: HUD句柄

## 消息管理

### `hideToast()`
- **功能**: 隐藏消息提示

## 拖拽视图管理

### `removeDragView(handle)`
- **功能**: 移除可拖动视图
- **参数**:
  - `handle`: 视图句柄

### `setDragViewOnClick(handle,callback)`
- **功能**: 设置可拖动视图点击回调
- **参数**:
  - `handle`: 视图句柄
  - `callback`: 点击回调函数
