# 梦幻繁星岛模块说明文档

## 1. 模块概述

梦幻繁星岛模块用于自动化完成《冲呀！饼干人：王国》中「梦幻繁星岛」活动的每日流程：从王国首页进入活动，经航海手册进入纯香草小岛完成每日签到，再在任务页领取可领奖奖励。模块设计为**每日执行一次**，当日重复调度时直接跳过。

---

## 2. 目录结构

```
帅斌饼干/脚本/game/常规_梦幻繁星岛/
├── 繁星岛_坐标库.lua   # UI 特征与点击坐标
├── 繁星岛_页面.lua     # 页面识别与交互 API
├── 繁星岛_路由.lua     # 导航：王国 → 梦幻繁星岛首页
├── 繁星岛_任务.lua     # 状态机任务编排
└── 繁星岛_会话.lua     # 今日是否已执行等持久化状态
```

---

## 3. 坐标库分区（繁星岛_坐标库.lua）

| 分区 | 说明 |
|---|---|
| `home` | 梦幻繁星岛首页特征、航海手册按钮、返回按钮、任务按钮 |
| `航海手册` | 航海手册页特征、登陆回忆小岛按钮 |
| `纯香草小岛` | 纯香草小岛页特征、返回按钮 |
| `任务` | 任务页特征、可领奖找色区域、返回按钮 |

---

## 4. 业务流程

完整 Mermaid 流程图见：`项目说明文档/梦幻繁星岛模块流程图.md`

简要步骤：

1. 判断今日是否已执行 → 是则 `done`
2. `detect` 识别页面 → `navigate` 从王国首页进入
3. 首页 → 航海手册 → 纯香草小岛 → 返回首页（完成每日签到）
4. 进入任务页 → 查找可领奖按钮
5. 有可领奖：点击领奖；无可领奖：标记今日已执行
6. 返回首页 → `done`

## 5. 调度接入

已完成接入：

1. `繁星岛_会话.lua` 提供 `isDoneToday()` / `markDoneToday()` / `clear()` / `describe()`。
2. `繁星岛_任务.lua` 按流程图实现状态机，状态包括：
   - `check` → `detect` → `navigate` → `openManual` → `enterIsland` → `returnFromIsland` → `openTask` → `claimTask` → `finish`
3. `game/register.lua` 已注册 `Scheduler.add("梦幻繁星岛", ...)`。
   - 优先级：矿山任务优先；矿山空闲时，若功能开启且今日未完成则执行。
   - 执行前会先离开布谷鸟广场（如果在广场上下文）。
4. `config.lua` 的 `STATIC.USER.starlight.enabled` 提供默认开关（默认 `false`）。
5. `ui/tabs/feature-tab.lua` 已增加"梦幻繁星岛"复选框开关，以及状态刷新/重置按钮。
